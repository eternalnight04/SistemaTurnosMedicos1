## Introducción

Este proyecto tiene como objetivo desarrollar el diseño orientado a objetos de una aplicación para administrar turnos médicos en
un hospital. La solución busca resolver que los turnos no se puedan agendar dentro del horario de otros ya agendados, que se pueda
avisar al paciente en caso de que el turno haya sido cancelado, poder tener una agenda activa donde guardar los turnos junto con su
correspondiente horario, fecha, doctor y paciente, entre otros beneficios que no podrían ser posibles si el sistema se siguiera
administrando con papel y lápiz. La idea es que facilite el trabajo de la secretaria y los profesionales del consultorio, siendo vital
también que el sistema sea lo más simple y fácil de entender posible.

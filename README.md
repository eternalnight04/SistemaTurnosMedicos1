# SistemaTurnosMedicos1

Este sistema para pedir turnos médicos busca que tenga una interfaz simple y amigable para cualquiera, que tanto la secretaria como los
profesionales puedan modificar o reprogramar turnos de manera rápida y fácil. Será una buena forma para dejar atrás el programado de turnos
de forma manual a lapicera y que el consultorio se pueda actualizar a un sistema digital para que todos puedan tener acceso y sea más
sencillo de manejar, al igual que para evitar que hayan problemas humanos como sugerir turnos que superpongan otros horarios y demás.
